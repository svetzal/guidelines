# Aspects of Code Quality

## Tools Assessment

What is being used for:

- code linting, readability
- testing
	- code coverage eg pytest-cov
	- mocking eg pytest-mock
	- special language features like async eg pytest-async
- project and project metadata management eg uv, maven, gradle, npm, cargo
- code security static analysis eg bandit, sobelow, clippy
- dependency security eg pip-audit, mix_audit, cargo-audit, npm audit
- dependency vulnerabilities and supply-chain attacks eg bandit, mix_audit, sobelow, cargo-deny, npm audit
- documentation eg mkdocs, ex_docs, mkdBook, vitepress
